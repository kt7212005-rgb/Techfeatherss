# Background Image

Place your desired background image in this folder and name it `bg.jpg`.

The app's CSS uses this image as the fixed background:

```
background: linear-gradient(135deg, rgba(238, 242, 243, 0.92) 0%, rgba(142, 158, 171, 0.92) 100%),
  url('../images/bg.jpg') center/cover fixed no-repeat;
```

If you want to use a different filename, update `assets/css/style.css` accordingly.
